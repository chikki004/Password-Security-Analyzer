Password Security Analyzer

A GUI-based Password Security Analyzer built using Python (Tkinter).

Features

- Password strength detection (Weak / Medium / Strong)
- Show / Hide password functionality
- Missing password requirement suggestions
- Entropy calculation (password randomness)
- Crack time estimation (brute-force simulation)
- Common password detection
- Strength progress bar visualization

Technologies Used

- Python
- Tkinter
- Regex
- Math
